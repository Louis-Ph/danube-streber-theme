# Support

This project is public and published on the VS Code Marketplace.

## Support Path

- GitHub issues are the main support channel:
  https://github.com/Louis-Ph/danube-streber-theme/issues
- Marketplace Q&A stays disabled until that workflow is live and maintained.

## Useful Issue Details

When reporting a visual problem, include:

- the theme variant: `Streber Dark`, `Streber Smoked Gold`, or `Streber Light`
- the language and file type involved
- whether the issue is in the editor, Markdown preview, terminal, diff, or suggestion widget
- a screenshot when practical
- whether the issue reproduces with the local files in `resources/language-samples/`
