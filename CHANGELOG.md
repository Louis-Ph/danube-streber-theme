## [1.0.0] - 2026-03-27

### Added
- Initial release of Streber Dark Theme
- Military camouflage green color palette
- Inspired by Danube streber fish colors
- Accessible contrast (WCAG AA)
- Full syntax highlighting for major languages:
  - JavaScript/TypeScript
  - Python
  - Rust
  - Go
  - Java
  - C/C++
  - HTML/CSS
  - Markdown
  - JSON/YAML
  - Shell scripts
- Custom icon with streber fish design
- Gallery banner with theme colors

### Changed
- Blue-gray and violet tones refined for elegance
- Inspired by French luxury fishing tackle aesthetics
- Optimized for reduced eye strain

### Technical
- VS Code extension packaged as .vsix
- Prepared for GitHub and Marketplace publication
