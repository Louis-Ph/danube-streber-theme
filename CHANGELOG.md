# Changelog

All notable changes to the Streber Dark Theme extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.7] - 2026-03-28

### Changed
- Repositioned the Marketplace README with stronger product messaging around the Streber theme ecosystem
- Refreshed the `Streber Light` preview image used in the public listing
- Updated the extension summary in `package.json` to match the new Marketplace framing

## [1.0.6] - 2026-03-28

### Changed
- Polished `Streber Light` across workbench colors and syntax scopes for a cleaner daytime profile
- Updated the Markdown documentation to reflect the refined light palette and current packaging workflow

## [1.0.5] - 2026-03-28

### Changed
- Refined `Streber Light` contrast in Markdown previews so inline and preformatted code no longer render with dark capsules
- Refreshed the dedicated `Streber Light` screenshot used in the public documentation

## [1.0.4] - 2026-03-28

### Added
- `Streber Light`, an experimental light variant on the `codex/streber-light` branch
- Dedicated Marketplace screenshot for `Streber Light`

### Changed
- Branch metadata and README now expose the three-theme trial set
- `SCREENSHOTS.md` was translated to English
- `SUPPORT.md` and `PUBLISHING.md` now reflect the current public release workflow

## [1.0.3] - 2026-03-28

### Added
- Explicit VSIX file allowlist in `package.json` to keep the published package minimal
- `SUPPORT.md` with a defined pre-public support path
- Semantic highlighting enabled in the theme manifest
- `Streber Smoked Gold`, a second dark theme variant with warmer brass and smoked-brown tones
- Dedicated Marketplace screenshot for `Streber Smoked Gold`

### Changed
- README now only references assets and links that actually exist
- Publication notes now reflect the real state of the project
- Extension metadata and README now describe the package as a two-theme collection
- The Marketplace README now presents both themes with a cleaner marketing layout
- Screenshot asset naming is aligned with `Streber Smoked Gold`

## [1.0.2] - 2026-03-28

### Changed
- Polished the Marketplace listing copy for clearer positioning and install instructions
- Aligned the README title and wording with the published extension name
- Kept a prominent screenshot while simplifying the page structure

## [1.0.1] - 2026-03-28

### Added
- README screenshot for the published Marketplace listing

### Changed
- Corrected Marketplace links and install commands in the README
- Aligned the published package metadata with the public GitHub repository

## [1.0.0] - 2026-03-27

### Added
- Initial release of Streber Dark Theme
- Military camouflage green color palette
- Inspired by Danube streber fish colors
- Accessible contrast (WCAG AA)
- Full syntax highlighting for major languages:
  - JavaScript/TypeScript
  - Python
  - Rust
  - Go
  - Java
  - C/C++
  - HTML/CSS
  - Markdown
  - JSON/YAML
  - Shell scripts
- Custom icon with streber fish design
- Gallery banner with theme colors

### Changed
- Blue-gray and violet tones refined for elegance
- Inspired by French luxury fishing tackle aesthetics
- Optimized for reduced eye strain

### Technical
- VS Code extension packaged as .vsix
- Prepared for GitHub and Marketplace publication
