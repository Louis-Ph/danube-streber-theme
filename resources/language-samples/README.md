# Language Samples

This folder contains one file per language for visual theme checks across the Streber theme collection.

## Suggested Review Flow

1. Open this folder in VS Code.
2. Switch to the theme under test.
3. Review comments, keywords, types, properties, methods, strings, numbers, operators, markup, and punctuation.
4. Trigger completion inside several files to inspect the suggestion widget.
5. Check one file with semantic highlighting enabled and one without, if the language extension allows both.
6. Review Markdown preview from `16-markdown.md` when validating `Streber Light`.

## Recommended Focus Areas

- `01-typescript.ts`, `02-javascript.js`: functions, properties, imports, decorators, completion
- `04-java.java`, `05-cpp.cpp`, `06-go.go`, `08-csharp.cs`: types, methods, control flow
- `09-rust.rs`, `10-ocaml.ml`, `11-bash.sh`: punctuation density, comments, operators
- `12-html.html`, `13-css.css`, `14-scss.scss`, `15-xml.xml`: tags, attributes, property names, units
- `16-markdown.md`: inline code, fenced blocks, links, quotes, preview rendering
- `17-json.json`, `18-yaml.yaml`: keys, values, punctuation, indentation cues
