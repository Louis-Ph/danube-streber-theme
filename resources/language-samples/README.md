# Language Samples

This folder contains one file per language for visual theme checks.

## Suggested Review Flow

1. Open this folder in VS Code.
2. Switch to the theme under test.
3. Review comments, keywords, types, functions, strings, numbers, operators, markup, and punctuation.
4. Trigger completion inside several files to inspect the suggestion widget.
